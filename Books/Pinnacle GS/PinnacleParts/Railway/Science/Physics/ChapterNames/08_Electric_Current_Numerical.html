✅ Correct Answer: (B)
  • The magnetic field produced by a current-carrying wire at a given point depends directly on the current passing through it.
  • If there is a circular coil having "n" turns, the field produced is "n" times as large as that produced by a single turn.
  • B = n(μ₀I/2R) (Where B = magnetic field, μ₀ = permeability, I = current and R = radius, n = turns).
  • B1/B2 = n/1, so the ratio is n : 1.
  • किसी बिंदु पर धारावाही तार द्वारा उत्पन्न चुंबकीय क्षेत्र उसमें से गुजरने वाली धारा पर सीधे निर्भर करता है।

  • B = n(μ₀I/2R) (जहाँ B = चुंबकीय क्षेत्र, μ₀ = चुंबकशीलता, I = धारा और R = त्रिज्या, n = फेरों की संख्या)।
  • B1/B2 = n/1, इसलिए अनुपात n : 1 है।

When a number of resistances are connected in __________ , their combined resistance is less than the smallest individual resistance.
