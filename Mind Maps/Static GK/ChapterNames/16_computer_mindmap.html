  • Booting: starting or restarting a computer
  • Window Task Manager: utility application of MS Windows
  • Standby: power management feature of Windows

TypeCoverageSpeedLocation
LAN1 Mile10 Mbps – 100 MbpsSchool, office, campus
MANUp to 50 km100 Mbps – 1 GbpsCity or metropolitan area

  • WAN uses High Level Data Link Control (HDLC) protocol.
  • Network Devices: Repeater, Gateway, Switch, Router, Modem, Hub
  • VPN: creates secure, encrypted connection between device and internet
  • Network Topology: Ring, Bus, Tree, Star, Mesh
  • ISDN: digital/standard telephone service
  • Cloud Computing: connects millions of mobile and personal computers via network
Internet
  • Internet: network of networks connecting computers worldwide
  • Connections: Dial-up; Broadband (cable modem, fiber optic, DSL); Wireless (Wi-Fi, Wi-MAX, satellite)
  • World Wide Web (WWW): created by Tim Berners-Lee in 1989
  • Web Page: documents written in HTML
  • Web Address: URL — indicates internet address of files
  • Domain Name: identifies a website (e.g., www.example.com)
  • Search Engines: Google, Yahoo, AltaVista, Lycos
